# 💰 Система напоминаний об оплате абонемента

## ✅ Реализовано: 10.11.2025

---

## 📋 Описание функционала

Добавлена система автоматических напоминаний для родителей о необходимости продления абонемента при остатке **≤ 2 занятий**.

---

## 🎯 Триггер напоминаний

### Условие показа:
```typescript
child.remainingClasses <= 2
```

### Уровни срочности:

| Осталось занятий | Статус | Цвет | Действие |
|-----------------|--------|------|----------|
| **0 занятий** | Закончился | 🔴 Красный (#D14343) | Срочно! Блокировка занятий |
| **1-2 занятия** | Заканчивается | 🟡 Золотой (#D4AF37) | Рекомендация продлить |
| **3+ занятий** | Активен | 🟢 Зеленый (#1C8C64) | Норма |

```typescript
  child => child.remainingClasses <= 2
);

// Уровни срочности

const isUrgent = child.remainingClasses === 0;
```

### 1. Автоматическая проверка при входе:
```typescript
// При загрузке ParentDashboard
useEffect(() => {
  const checkSubscriptions = async () => {
    const children = await fetchChildren(userId);
    const needRenewal = children.filter(c => c.remainingClasses <= 2);
    
    if (needRenewal.length > 0) {
      // Показать уведомление
      toast.warning(`У ${needRenewal.length} детей заканчивается абонемент`);
    }
  };
  
  checkSubscriptions();
}, [userId]);
```

### 2. Push-уведомления:
```typescript
// При остатке 2 занятий - отправить push
if (child.remainingClasses === 2) {
  sendPushNotification(parentId, {
    title: 'Напоминание об оплате',
    body: `У ${child.name} осталось 2 занятия`
  });
}
```

### 3. Email-напоминания:
```typescript
// При остатке 2 и 0 занятий
if ([2, 0].includes(child.remainingClasses)) {
  sendEmail(parent.email, {
    subject: 'Продлите абонемент',
    template: 'subscription-renewal',
    data: { childName, remainingClasses }
  });
}
```

### 4. SMS-уведомления:
```typescript
// При остатке 0 занятий (срочно)
```
