# 🔄 Changelog: Добавление кнопок оплаты на уведомления

**Дата:** 10 ноября 2025  
**Версия:** 1.1  
**Автор:** AI Assistant  

---

## 📋 Краткое описание изменений

Добавлены кнопки оплаты на все уведомления о необходимости продления абонемента для родителей. Теперь каждое напоминание имеет прямые кнопки действия для оплаты.

---

## ✨ Новый функционал

### 1. Кнопки в Alert блоках

**Где:** ParentHome & ParentChildren  
**Что добавлено:**
- ✅ Основная кнопка "Оплатить абонемент" (с иконкой CreditCard)
- ✅ Дополнительная кнопка "Выбрать другой план"

**Пример:**
```tsx
<div className="flex flex-wrap gap-2">
  <Button className="bg-gradient-to-r from-[#133C2A] to-[#D4AF37]">
    <CreditCard className="w-4 h-4 mr-2" />
    Оплатить абонемент
  </Button>
  <Button variant="outline">
    Выбрать другой план
  </Button>
</div>
```

### 2. Кнопки на карточках детей

**Где:** ParentHome & ParentChildren  
**Что добавлено:**
- ✅ Условная кнопка (показывается только при `remainingClasses <= 2`)
- ✅ Адаптивный текст: "Продлить абонемент" / "Срочно продлить"
- ✅ Адаптивный цвет: золотой градиент / красно-золотой градиент

**Логика:**
```tsx
{needsRenewal && (
  <Button className={
    isUrgent 
      ? 'bg-gradient-to-r from-[#D14343] to-[#D4AF37]'  // Красный
      : 'bg-gradient-to-r from-[#133C2A] to-[#D4AF37]'  // Обычный
  }>
    <CreditCard className="w-4 h-4 mr-2" />
    {isUrgent ? 'Срочно продлить' : 'Продлить абонемент'}
  </Button>
)}
```

   ```tsx
   onClick={() => navigate('/parent/payments')}
   ```

2. **Открытие диалога выбора абонемента:**
   ```tsx
   onClick={() => setShowSubscriptionDialog(true)}
   ```

3. **Прямой переход к оплате:**
   ```tsx
   onClick={() => handlePayment(child.id)}
   ```

4. **Предзаполнение формы:**
   ```tsx
   onClick={() => {
     setSelectedChild(child);
     setSelectedSubscription(child.subscriptionId);
     navigate('/parent/payments');
   }}
   ```
